# Digital Dashboard

A sleek, modern dashboard featuring:

- **Random nature background** 
- **Dogecoin price** 
- **Live clock**
- **Local weather** 
- **Rotating Pulp Fiction quotes** with natural background

---

## Features

| Feature              | Source                     | Update Interval |
|----------------------|----------------------------|-----------------|
| Background Photo     | Unsplash                   | On load         |
| Dogecoin Price       | CoinGecko                  | On load         |
| Clock                | Local                      | 1 second        |
| Weather              | OpenWeatherMap             | On load         |
| **Pulp Fiction Quote** | Hardcoded (10 quotes)    | **20 seconds**  |

---

### Tech Stack

- HTML5
- CSS3 (Glassmorphism + Flexbox)
- Vanilla JavaScript (Async/Await, Fetch API)
- Free Public APIs
